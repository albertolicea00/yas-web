# Yet Another Store (YAS) Suite

> A collection of modern, premium, and unified graphical user interfaces (GUIs) for CLI package managers across macOS, Windows, and Linux.

---

## 🚀 Welcome to YAS

Command line package managers are incredibly powerful, but we all suffer from **command amnesia**. Who hasn't spent five minutes searching help docs or StackOverflow just to remember the exact syntax to pin a package, clean a cache, or list unneeded dependencies?

**Yet Another Store (YAS)** brings all that CLI power into stunning, native-feeling graphical user interfaces. No more forgotten commands, no more syntax errors. Just pure, visual package management.

---

## ✨ Features

- **Visual Package Explorer** – Search, filter, and inspect package metadata with ease.
- **One‑Click Management** – Install, update, pin, and uninstall packages without opening the terminal.
- **Smart Command Logs** – See the exact commands run under the hood (great for learning and transparency).
- **Beautiful Unified Icons** – Every store comes with unique, high‑quality front‑store icon variants representing its brand identity.
